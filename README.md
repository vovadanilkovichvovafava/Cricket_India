# 🏏 Cricket Bet Analyzer — IPL 2026

AI-powered cricket betting analysis platform for IPL.

## Quick Start

```bash
# Install
cd web && npm install

# Dev
npm run dev          # → localhost:5173

# Full stack with Remote Control + Cloudflare Tunnel
./start-dev.sh
```

## Stack
- **Frontend:** React + Vite + Tailwind
- **Backend:** Python FastAPI + Claude AI + XGBoost
- **Deploy:** Railway + Docker
- **Dev workflow:** Claude Code Remote Control + Cloudflare Tunnel
